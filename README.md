# Gym Landing Page

Introducing our Gym Landing Page, a dynamic and engaging platform for your fitness journey. This project, developed with TypeScript, React, and Vite, offers a smooth and efficient user experience. With TypeScript and React, you can create a responsive and feature-rich landing page. Tailwind CSS ensures a stylish and customizable design, Hero Icons provides a library of sleek icons, and Framer Motion adds captivating animations. This project was inspired by a YouTube video tutorial by Edroh.
